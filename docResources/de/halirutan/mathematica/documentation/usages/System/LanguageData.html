LanguageData

  • LanguageData["name","property"] gives the value of the specified property for the language "name".
  • LanguageData[{name1,name2, ... },"property"] gives a list of property values for the specified language names.
  • LanguageData["name", "property", annotation] gives the specified annotation associated with the property.

Attributes:Protected, ReadProtected

Symbol has no options.
