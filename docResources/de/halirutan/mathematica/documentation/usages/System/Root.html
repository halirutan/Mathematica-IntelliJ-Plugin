Root

  • Root[f,k] represents the exact kth root of the polynomial equation f[x] ⩵ 0.
  • Root[{f1,f2, … },{k1,k2, … }] represents the last coordinate of the exact vector {a1,a2, … } such that ai is the kith root of the polynomial equation fi[a1, … ,ai-1,x] ⩵ 0.
  • Root[{f,x0}] represents the exact root of the general equation f[x] ⩵ 0 near x=x0.
  • Root[{f,x0,n}] represents n roots of the equation f[x] ⩵ 0 near x=x0.

Attributes: NHoldAll, Protected, ReadProtected

Options: ExactRootIsolation
