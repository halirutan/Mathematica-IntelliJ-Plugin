NDSolve

  • NDSolve[eqns,u,{t,tmin,tmax}] finds a numerical solution to the ordinary differential equations eqns for the function u with the independent variable t in the range tmin to tmax.
  • NDSolve[eqns,u,{t,tmin,tmax},{x,xmin,xmax}] finds a numerical solution to the partial differential equations eqns.
  • NDSolve[eqns,{u1,u2, … },{t,tmin,tmax}] finds numerical solutions for the functions   ui.

Attributes: Protected

Options: AccuracyGoal, Compiled, DependentVariables, DiscreteVariables, EvaluationMonitor, InterpolationOrder, MaxStepFraction, MaxSteps, MaxStepSize, Method, NormFunction, PrecisionGoal, StartingStepSize, StepMonitor, WorkingPrecision
