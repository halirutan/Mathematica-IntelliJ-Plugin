GetEnvironment

  • GetEnvironment["var"] gives the setting corresponding to the variable "var" in the operating system environment.
  • GetEnvironment[{"StyleBox[RowBox[{var, StyleBox[1, FontSlant -> Plain]}]]","StyleBox[RowBox[{var, StyleBox[2, FontSlant -> Plain]}]]", … }] gives a list of rules, corresponding to specified environment variables.
  • GetEnvironment[] gives information about all existing settings in the operating system environment.

Attributes: Protected

Symbol has no options.
