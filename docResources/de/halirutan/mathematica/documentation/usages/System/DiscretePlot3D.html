DiscretePlot3D

  • DiscretePlot3D[expr,{i,imin,imax},{j,jmin,jmax}] generates a plot of the values of expr when i runs from imin to imax and j runs from jmin to jmax.
  • DiscretePlot3D[expr,{i,imin,imax,di},{j,jmin,jmax,dj}] uses steps di and dj.
  • DiscretePlot3D[expr,{i,{i1,i2, … }},{j,{j1,j2, … }}] uses successive i values i1, i2, … and j values j1, j2, … .
  • DiscretePlot3D[{expr1,expr2, … }, … , … ] plots the values of all the expri.
