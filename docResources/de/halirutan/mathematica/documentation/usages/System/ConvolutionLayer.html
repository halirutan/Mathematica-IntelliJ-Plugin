ConvolutionLayer

  • ConvolutionLayer[n,s] represents a trainable 2D convolutional net layer having n output channels, and uses s × s kernels to compute the convolution.
  • ConvolutionLayer[n,{h,w}] uses kernels of size h × w.
  • ConvolutionLayer[n,{h,w},opts] includes options for initial kernels and other parameters.

Attributes:Protected, ReadProtected

Symbol has no options.
