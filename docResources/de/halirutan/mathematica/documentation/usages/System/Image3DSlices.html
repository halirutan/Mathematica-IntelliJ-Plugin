Image3DSlices

  • Image3DSlices[image] gives a list of 2D images corresponding to the slices in the Image3D object image.
  • Image3DSlices[image,{s1,s2, … }] extracts the specified slices si
  • Image3DSlices[image,{s1,s2, … },d] takes slices in dimension d.

Attributes: Protected, ReadProtected

Symbol has no options.
