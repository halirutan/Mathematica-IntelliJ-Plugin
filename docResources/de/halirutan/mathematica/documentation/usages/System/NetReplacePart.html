NetReplacePart

  • NetReplacePart[layer,"array" → value] replaces an array within a layer, returning a new layer.
  • NetReplacePart[layer,"port" → type] returns a new layer in which an input or output port has the specified type.
  • NetReplacePart[net,"input" → encoder] attaches a NetEncoder[ … ] to a specified input port.
  • NetReplacePart[net,"output" → decoder] attaches a NetDecoder[ … ] to a specified output port.
  • NetReplacePart[net,{lspec,pspec} → value] makes a replacement of a part pspec of a layer lspec within a NetGraph or NetChain.
  • NetReplacePart[net,{spec1 → val1,spec2 → val2, … }] makes multiple simultaneous replacements.

Attributes:Protected, ReadProtected

Symbol has no options.
