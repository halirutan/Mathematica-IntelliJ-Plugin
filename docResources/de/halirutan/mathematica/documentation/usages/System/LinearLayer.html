LinearLayer

  • LinearLayer[n] represents a trainable, fully connected net layer that computes w.x+b with output vector of size n.
  • LinearLayer[{n1,n2, … }] represents a layer that outputs a tensor of dimensions n1 × n2 × … .
  • LinearLayer[] leaves the dimensions of the output tensor to be inferred from context.
  • LinearLayer[n,opts] includes options for initial weights and other parameters.

Attributes:Protected, ReadProtected

Symbol has no options.
