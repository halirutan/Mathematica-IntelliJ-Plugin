CloudObject

  • CloudObject[] represents a new anonymous cloud object.
  • CloudObject[http://StyleBox[ ... , TR]], CloudObject[https://StyleBox[ ... , TR]] represents a cloud object with a given URI.
  • CloudObject[user:StyleBox[user, TI]/StyleBox[path, TI]] represents a cloud object of a given user with a given path.
  • CloudObject[/StyleBox[abspath, TI]] represents a cloud object of the currently authenticated user at an absolute path.
  • CloudObject["relpath"] represents a cloud object of the currently authenticated user at a relative path.
  • CloudObject["relpath",cbase] represents a cloud object with a path relative to the base cbase.

Attributes:Protected, ReadProtected

Symbol has no options.
