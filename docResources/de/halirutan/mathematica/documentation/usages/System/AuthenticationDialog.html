AuthenticationDialog

  • AuthenticationDialog[] initiates a standard dialog for entering username/password authentication information.
  • AuthenticationDialog["type"] initiates an authentication dialog of the specified standard type.
  • AuthenticationDialog[{key1,key2, … }] initiates an authentication dialog that requests values for the specified keys.
  • AuthenticationDialog[arg,func] applies the function func to the dialog's return value.

Attributes:Protected, ReadProtected

Symbol has no options.
