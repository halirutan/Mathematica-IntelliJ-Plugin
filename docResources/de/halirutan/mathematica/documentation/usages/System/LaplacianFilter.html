LaplacianFilter

  • LaplacianFilter[image,r] convolves image with a range-r Laplacian kernel.
  • LaplacianFilter[image,{r1,r2}] uses ranges ri in the vertical and horizontal directions.
  • LaplacianFilter[data, … ] applies Laplacian filtering to an array of data.

Attributes: Protected, ReadProtected

Options: Padding, WorkingPrecision
