Image3D

  • Image3D[data] represents a 3D image with pixel values given by the array data.
  • Image3D[{image1,image2, … }] creates a 3D image from a list of 2D images.
  • Image3D[obj,"type"] creates a 3D image of the specified data type.

Attributes: Protected, ReadProtected

Options: AlignmentPoint, Axes, AxesEdge, AxesLabel, AxesOrigin, AxesStyle, Background, BaselinePosition, Boxed, BoxRatios, BoxStyle, ClipRange, ColorFunction, ColorSpace, CoordinatesToolOptions, ImageResolution, ImageSize, Interleaving, Magnification, MetaInformation, Method, RotationAction, SphericalRegion, TaggingRules, ViewAngle, ViewCenter, ViewMatrix, ViewPoint, ViewRange, ViewVector, ViewVertical
