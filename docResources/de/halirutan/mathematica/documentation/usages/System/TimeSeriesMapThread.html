TimeSeriesMapThread

  • TimeSeriesMapThread[f,tseries] gives {{t1,f[t1,x1]},{t2,f[t2,x2]}, ... } for the time series tseries.
  • TimeSeriesMapThread[f,tseries,{{a1,a2, ... },{b1,b2, ... }, ... }] gives {f[t1,x1,a1,b1, ... ],f[t2,x2,a2,b2, ... ], ... } for the time series tseries.

Attributes:Protected, ReadProtected

Symbol has no options.
