HistoricalPeriodData

  • HistoricalPeriodData["name","property"] gives the value of the specified property for the historical period "name".
  • HistoricalPeriodData[{name1,name2, ... },"property"] gives a list of property values for the specified historical period names.
  • HistoricalPeriodData["name", "property", annotation] gives the specified annotation associated with the property.

Attributes:Protected, ReadProtected

Symbol has no options.
