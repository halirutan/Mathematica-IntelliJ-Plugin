ARIMAProcess

  • ARIMAProcess[{a1, … ,ap},d,{b1, … ,bq},v] represents the autoregressive integrated moving-average process y(t) such that its dth difference is an ARMAProcess[{a1, … ,ap},{b1, … ,bq},v].
  • ARIMAProcess[{a1, … ,ap},d,{b1, … ,bq}, Σ ] represents the vector autoregressive integrated moving-average process (y1(t), … ,yn(t)) such that its (d, … ,d)th difference is a vector ARMAProcess.
  • ARIMAProcess[{a1, … ,ap},{d1, … ,dn},{b1, … ,bq}, Σ ] represents the vector autoregressive integrated moving-average process (y1(t), … ,yn(t)) such that its (d1, … ,dn)th difference is a vector ARMAProcess.

Attributes: Protected, ReadProtected
