SelectComponents

  • SelectComponents[m,"prop",crit] computes a property "prop" for all components in a label matrix m and returns a version of m in which components that do not satisfy crit are replaced with zeros.
  • SelectComponents[m,"prop",n] gives a label matrix for the first n components, ranked by property "prop".
  • SelectComponents[m,"prop",n,p] uses an ordering function p.

Attributes: Protected, ReadProtected

Options: CornerNeighbors
