GaborFilter

  • GaborFilter[image,r,k] filters image by convolving with a Gabor kernel of pixel radius r and wave vector k.
  • GaborFilter[image,r,k, ϕ ] uses a Gabor kernel with phase shift ϕ .
  • GaborFilter[image,{r, σ }, … ] uses a Gabor kernel with radius r and standard deviation σ .
  • GaborFilter[data, … ] applies Gabor filtering to an array of data.

Attributes: Protected, ReadProtected

Options: Standardization, Padding, WorkingPrecision
