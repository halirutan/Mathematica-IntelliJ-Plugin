ReplaceImageValue

  • ReplaceImageValue[image,{x,y}->val] changes the pixel values at position {x,y} in image to val.
  • ReplaceImageValue[image,{xmin;;xmax,ymin;;ymax}->val] changes the pixel values at positions that range from {xmin,ymin} to {xmax,ymax} to val.
  • ReplaceImageValue[image,{x,y}->val,"type"] assumes val to be of the specified type.

Attributes: Protected, ReadProtected

Options: DataRange
