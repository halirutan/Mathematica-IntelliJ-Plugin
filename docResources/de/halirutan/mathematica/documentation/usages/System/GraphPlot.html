GraphPlot

  • GraphPlot[{vi ​ 1->vj ​ 1,vi ​ 2->vj ​ 2, … }] generates a plot of the graph in which vertex vik is connected to vertex vjk.
  • GraphPlot[{{vi ​ 1->vj ​ 1,lbl1}, … }] associates labels lblk with edges in the graph.
  • GraphPlot[m] generates a plot of the graph represented by the adjacency matrix m.
