ConoverTest

  • ConoverTest[{data1,data2}] tests whether the variances of data1 and data2 are equal.
  • ConoverTest[dspec, σ 0 2 ] tests a dispersion measure against σ 0 2 .
  • ConoverTest[dspec, σ 0 2 ,"property"] returns the value of "property".

Attributes: Protected, ReadProtected

Options: Method, AlternativeHypothesis, SignificanceLevel, VerifyTestAssumptions
