CoordinateBoundingBox

  • CoordinateBoundingBox[coords] gives the corners {{xmin,ymin, … },{xmax,ymax, … }} of the bounding box of the region defined by coords.
  • CoordinateBoundingBox[coords, δ ] pads the region by δ in each direction.
  • CoordinateBoundingBox[coords,Scaled[s]] pads by the scaled amount s in each direction.
  • CoordinateBoundingBox[coords,{p1,p2, … }] pads by p1, p2, … in successive dimensions.
  • CoordinateBoundingBox[coords,{{p1min,p1max},{p2min,p2max}, … }] gives {{xmin-p1min,ymin-p2min, … },{xmax+p1max,ymax+p2max, … }, … }
