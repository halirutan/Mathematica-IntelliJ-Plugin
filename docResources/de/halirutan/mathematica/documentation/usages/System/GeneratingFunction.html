GeneratingFunction

  • GeneratingFunction[expr,n,x] gives the generating function in x for the sequence whose nth series coefficient is given by the expression expr.
  • GeneratingFunction[expr,{n1,n2, … },{x1,x2, … }] gives the multidimensional generating function in x1, x2, … whose n1, n2, … coefficient is given by expr.

Attributes: Protected, ReadProtected

Options: Assumptions, GenerateConditions, Method, VerifyConvergence
