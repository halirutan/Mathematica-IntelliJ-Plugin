ProbitModelFit

  • ProbitModelFit[{y1,y2, … },{f1,f2, … },x] constructs a binomial probit regression model of the form (1)/(2) (1+erf(( β 0+ β 1f1 + β 2f2 + ⋯ )√(2))) that fits the yi for successive x values 1, 2, … .
  • ProbitModelFit[{{x11,x12, … ,y1},{x21,x22, … ,y2}, … },{f1,f2, … },{x1,x2, … }] constructs a binomial probit regression model of the form (1)/(2) (1+erf(( β 0+ β 1f1 + β 2f2 + ⋯ )√(2))) where the fi depend on the variables xk.
  • ProbitModelFit[{m,v}] constructs a binomial probit regression model from the design matrix m and response vector v.

Attributes: Protected, ReadProtected

Options: AccuracyGoal, ConfidenceLevel, CovarianceEstimatorFunction, DispersionEstimatorFunction, IncludeConstantBasis, LinearOffsetFunction, MaxIterations, NominalVariables, PrecisionGoal, Weights, WorkingPrecision
