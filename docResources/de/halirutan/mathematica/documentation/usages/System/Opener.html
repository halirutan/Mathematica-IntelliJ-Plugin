Opener

  • Opener[x] represents an opener with setting x, displayed as Opener[True] when x is True and Opener[False] when x is False.
  • Opener[Dynamic[x]] takes the setting to be the dynamically updated current value of x, with the value of x being toggled if the opener is clicked.

Attributes: Protected, ReadProtected

Options: Appearance, AutoAction, Background, BaselinePosition, BaseStyle, ContentPadding, ContinuousAction, DefaultBaseStyle, Enabled, ImageMargins
