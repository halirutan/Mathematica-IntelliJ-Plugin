TemplateBox

  • TemplateBox[{box1,box2,...},tag] is a low-level box structure that parameterizes the display and evaluation of the boxes boxi.

Attributes: Protected, ReadProtected

Options: BaseStyle, DefaultBaseStyle, DefaultTooltipStyle, DisplayFunction, Editable, InterpretationFunction, Selectable, SyntaxForm, Tooltip, TooltipDelay, TooltipStyle
