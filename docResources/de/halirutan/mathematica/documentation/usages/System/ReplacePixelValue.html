ReplacePixelValue

  • ReplacePixelValue[image,{xp,yp}->val] changes the pixel values at pixel position {xp,yp} in image to val.
  • ReplacePixelValue[image,{xpmin;;xpmax,ypmin;;ypmax}->val] changes the pixel values at the pixel positions that range from {xpmin,ypmin} to {xpmax,ypmax} to val.
  • ReplacePixelValue[image,{xp,yp}->val,"type"] assumes val to be of the specified type.

Attributes: Protected, ReadProtected

Symbol has no options.
