InterpretationBox

  • InterpretationBox[boxes,expr] is a low-level box construct that displays as boxes but is interpreted on input as expr.

Attributes: HoldAllComplete, Protected, ReadProtected

Options: AutoDelete, BaseStyle, DefaultBaseStyle, DefaultTooltipStyle, DeleteWithContents, DeletionWarning, Editable, InterpretationNote, SelectWithContents, Selectable, SyntaxForm, Tooltip, TooltipDelay, TooltipStyle
