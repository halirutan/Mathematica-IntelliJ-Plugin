EquirippleFilterKernel

  • EquirippleFilterKernel[{{{ ω L1, ω R1},{ ω L2, ω R2}, … },{a1,a2, … }},n] creates a finite impulse response (FIR) filter kernel of length n with an equiripple amplitude response, given the specified left and right band edge frequencies { ω Li, ω Ri} and amplitudes ai.
  • EquirippleFilterKernel[{{{ ω L1, ω R1},{ ω L2, ω R2}, … },{a1,a2, … },{w1, … }},n] uses relative weights wi for each frequency band.
  • EquirippleFilterKernel[{"type",{{{ ω L1, ω R1}, … }, … },n] creates a filter of the specified "type".

Attributes: Protected, ReadProtected

Options: GridDensity, WorkingPrecision
