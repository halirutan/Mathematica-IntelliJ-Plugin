FrameBox

  • FrameBox[box] is a low-level box construct which represents box with a frame drawn around it.

Attributes: Protected, ReadProtected

Options: Alignment, Appearance, Background, BaseStyle, BaselinePosition, BoxExpandable, BoxFrame, ContentPadding, DefaultBaseStyle, DefaultFrameStyle, Enabled, FrameMargins, FrameStyle, ImageMargins, ImageSize, RoundingRadius, StripOnInput
