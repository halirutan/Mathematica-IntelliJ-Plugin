NonlinearModelFit

  • NonlinearModelFit[{y1,y2, … },form,{ β 1, … },x] constructs a nonlinear model with structure form that fits the yi for successive x values 1, 2, … using the parameters β 1, … .
  • NonlinearModelFit[{{x11,x12, … ,y1},{x21,x22, … ,y2}, … },form,{ β 1, … },{x1, … }] constructs a nonlinear model where form depends on the variables xk.
  • NonlinearModelFit[data,{form,cons},{ β 1, … },{x1, … }] constructs a nonlinear model subject to the parameter constraints cons.

Attributes: Protected, ReadProtected

Options: AccuracyGoal, ConfidenceLevel, EvaluationMonitor, Gradient, MaxIterations, Method, PrecisionGoal, StepMonitor, Tolerance, VarianceEstimatorFunction, Weights, WorkingPrecision
