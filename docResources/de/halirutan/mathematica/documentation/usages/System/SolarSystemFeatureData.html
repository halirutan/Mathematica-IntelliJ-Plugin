SolarSystemFeatureData

  • SolarSystemFeatureData["name","property"] gives the value of the specified property for the solar system feature "name".
  • SolarSystemFeatureData[{name1,name2, ... },"property"] gives a list of property values for the specified feature names.
  • SolarSystemFeatureData["name","property",annotation] gives the specified annotation associated with the property.

Attributes:Protected, ReadProtected

Symbol has no options.
