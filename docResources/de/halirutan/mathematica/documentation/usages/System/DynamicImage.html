DynamicImage

  • DynamicImage[image] displays a dynamic version of image, supporting panning, zooming, etc.
  • DynamicImage[file] displays a dynamic version of the image stored in file.
  • DynamicImage[url] displays a dynamic version of the image stored in url.

Attributes:HoldAllComplete, Protected, ReadProtected

Options: AppearanceElements, AspectRatio, BaselinePosition, Epilog, ImageSize, SaveDefinitions, ZoomCenter, ZoomFactor
