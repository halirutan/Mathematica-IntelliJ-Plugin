NArgMin

  • NArgMin[f,x] gives a position xmin at which f is numerically minimized.
  • NArgMin[f,{x,y, … }] gives a position {xmin,ymin, … } at which f is numerically minimized.
  • NArgMin[{f,cons},{x,y, … }] gives a position at which f is numerically minimized subject to the constraints cons.

Attributes: Protected, ReadProtected

Options: AccuracyGoal, EvaluationMonitor, MaxIterations, Method, PrecisionGoal, StepMonitor, WorkingPrecision
