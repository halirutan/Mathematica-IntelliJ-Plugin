SingularValuePlot

  • SingularValuePlot[lsys] generates a plot of the singular values of the transfer function for the system lsys.
  • SingularValuePlot[lsys,{ ω min, ω max}] plots for the frequency range ω min to ω max.
  • SingularValuePlot[expr,{ ω , ω min, ω max}] plots expr using the variable ω .
