StreamDensityPlot

  • StreamDensityPlot[{{vx,vy},s},{x,xmin,xmax},{y,ymin,ymax}] generates a stream plot of the vector field {vx,vy} as a function of x and y, superimposed on a background density plot of the scalar field s.
  • StreamDensityPlot[{vx,vy},{x,xmin,xmax},{y,ymin,ymax}] takes the scalar field to be the norm of the vector field.
  • StreamDensityPlot[{{vx,vy},{wx,wy}, … ,s},{x,xmin,xmax},{y,ymin,ymax}] generates plots of several vector fields.
