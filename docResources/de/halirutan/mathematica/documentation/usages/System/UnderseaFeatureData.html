UnderseaFeatureData

  • UnderseaFeatureData["name","property"] gives the value of the specified property for the undersea feature "name".
  • UnderseaFeatureData[{name1,name2, ... },"property"] gives a list of property values for the specified undersea feature names.
  • UnderseaFeatureData["name", "property", annotation] gives the specified annotation associated with the property.

Attributes:Protected, ReadProtected

Symbol has no options.
