FormulaData

  • FormulaData[name] gives the equations for the formula name.
  • FormulaData[name,{var1 → quantity1,var2 → quantity2, ... }] inserts the specified values quantityi for the variables vari into the equations for the formula.
  • FormulaData[name,"property"] gives the value of the specified property for the formula name.

Attributes:Protected, ReadProtected

Symbol has no options.
