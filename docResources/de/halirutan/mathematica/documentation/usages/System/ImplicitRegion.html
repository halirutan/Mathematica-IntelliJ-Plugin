ImplicitRegion

  • ImplicitRegion[cond,{x1, ... ,xn}] represents a region in  n that satisfies the conditions cond.
  • ImplicitRegion[cond,{{x1,a1,b1}, ... }] represents a region in  n that satisfies the conditions cond as well as a1 <= x1 <= b1 etc.
