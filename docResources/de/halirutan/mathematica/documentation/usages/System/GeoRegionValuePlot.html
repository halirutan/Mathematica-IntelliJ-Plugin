GeoRegionValuePlot

  • GeoRegionValuePlot[{reg1 → val1,reg2 → val2, ... }] generates a plot in which the geographic regions regi are colored according to the values vali.
  • GeoRegionValuePlot[reg → val] generates a plot in which the geographic subdivisions of reg are colored according to the values EntityValue[ ... ,val].
  • GeoRegionValuePlot[entityclass → val] generates a plot in which the geographic regions in entityclass are colored according to the values EntityValue[ ... ,val].
