ARProcess

  • ARProcess[{a1, … ,ap},v] represents an autoregressive process of order p with normal white noise that has variance v.
  • ARProcess[{a1, … ,ap}, Σ ] represents a vector autoregressive process with multinormal white noise that has covariance matrix Σ .
  • ARProcess[tproc,p] approximate a time series process tproc with an AR process of order p.
