DynamicWrapper

  • DynamicWrapper[e,expr] represents an object that displays as e, but dynamically updates the expression expr whenever that object is visible on screen.

Attributes: HoldRest, Protected, ReadProtected

Options: Appearance, AutoAction, Background, BaseStyle, CachedValue, ContinuousAction, DefaultBaseStyle, Deinitialization, DisplayFunction, Enabled, Evaluator, ImageSizeCache, Initialization, BoxForm`RecursionLimit, Selectable, ShrinkingDelay, SingleEvaluation, SynchronousUpdating, TrackedSymbols, UpdateInterval
