SystemModelSimulate

  • SystemModelSimulate[model] simulates model according to experiment settings.
  • SystemModelSimulate[model,tmax] simulates from 0 to tmax.
  • SystemModelSimulate[model,{tmin,tmax}] simulates from tmin to tmax.
  • SystemModelSimulate[model,vars,{tmin,tmax}] stores only simulation data for the variables vars.

Attributes:Protected, ReadProtected

Symbol has no options.
