MaximalBy

  • MaximalBy[{e1,e2, ... },f] returns a list of the ei for which the value of f[ei] is maximal.
  • MaximalBy[{e1,e2, ... },f,n] returns a list of the ei corresponding to the n largest f[ei].
  • MaximalBy[f] represents an operator form of MaximalBy that can be applied to an expression.

Attributes:Protected, ReadProtected

Symbol has no options.
