InitializationValue

  • InitializationValue[sym] represents the settable persistent value with which the symbol sym will be initialized.
  • InitializationValue["context`name"] represents the settable initialization value for the symbol with the specified complete name.
  • InitializationValue[sym,{loc1, … }] specifies the persistence locations to search for a possible initialization value.

Attributes:HoldFirst, Protected, ReadProtected

Options: EvaluationEnvironment, MergingFunction, ValuePreprocessingFunction, ExpirationDate, PersistenceTime
