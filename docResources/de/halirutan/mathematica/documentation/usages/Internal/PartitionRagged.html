PartitionRagged

  • PartitionRagged[list, {n1,...,nk}] partitions list into ragged array with rows of
  • length n1, ..., nk. PartitionRagged[ array, {{n11,...,},...,{nm1,...}}] partitions
  • depth m array along each dimension. PartitionRagged[list, ns, fn] maps a function
  • fn to the partitioned list.
