Hyperlink

  • Hyperlink[uri] represents a hyperlink that jumps to the specified URI when clicked.
  • Hyperlink[label,uri] represents a hyperlink to be displayed as label.

Attributes: Protected, ReadProtected

Options: Active, ActiveStyle, Alignment, Appearance, AutoAction, Background, BaselinePosition, BaseStyle, ContentPadding, DefaultBaseStyle, Enabled, Evaluator, FrameMargins, ImageMargins, ImageSize, Method
