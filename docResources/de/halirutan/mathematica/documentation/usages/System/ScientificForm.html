ScientificForm

  • ScientificForm[expr] prints with all real numbers in expr given in scientific notation.
  • ScientificForm[expr,n] prints with numbers given to n ‐ digit precision.

Attributes: Protected

Options: DigitBlock, ExponentFunction, ExponentStep, NumberFormat, NumberMultiplier, NumberPadding, NumberPoint, NumberSeparator, NumberSigns, SignPadding
