VarianceTest

  • VarianceTest[data] tests whether the variance of the data is one.
  • VarianceTest[{data1,data2}] tests whether the variances of data1 and data2 are equal.
  • VarianceTest[dspec, σ 02] tests a dispersion measure against σ 02.
  • VarianceTest[dspec, σ 02,"property"] returns the value of "property".

Attributes: Protected, ReadProtected

Options: Method, AlternativeHypothesis, SignificanceLevel, VerifyTestAssumptions
