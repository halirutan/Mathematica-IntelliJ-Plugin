BattleLemarieWavelet

  • BattleLemarieWavelet[] represents the Battle-Lemari é wavelet of order 3.
  • BattleLemarieWavelet[n] represents the Battle-Lemari é wavelet of order n evaluated on equally spaced interval {-10,10}.
  • BattleLemarieWavelet[n,lim] represents the Battle-Lemari é wavelet of order n evaluated on equally spaced interval {-lim,lim}.

Attributes: Protected, ReadProtected

Symbol has no options.
