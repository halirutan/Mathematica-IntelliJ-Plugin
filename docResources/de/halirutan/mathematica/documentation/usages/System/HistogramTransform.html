HistogramTransform

  • HistogramTransform[image] transforms pixel values of image so that its histogram is nearly flat.
  • HistogramTransform[image,dist] modifies pixel values of image so that its histogram matches the probability density function (PDF) of the symbolic distribution dist.
  • HistogramTransform[image,ref] matches the histogram of image with the histogram of the reference image ref.
  • HistogramTransform[{x1,x2, … }, … ] transforms values xi.

Attributes: Protected, ReadProtected

Symbol has no options.
