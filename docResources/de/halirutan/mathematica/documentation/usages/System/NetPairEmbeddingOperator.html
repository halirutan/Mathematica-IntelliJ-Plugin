NetPairEmbeddingOperator

  • NetPairEmbeddingOperator[net] represents a net that takes a pair of tensors, embeds them into a vector space using net, and outputs the distance under the embedding.
  • NetPairEmbeddingOperator[net,"DistanceFunction" → f] specifies a distance function to use.

Attributes:Protected, ReadProtected

Symbol has no options.
