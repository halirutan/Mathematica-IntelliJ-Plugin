NicholsPlot

  • NicholsPlot[lsys] generates a Nichols plot of the transfer function for the system lsys.
  • NicholsPlot[lsys,{ ω min, ω max}] plots for the frequency range ω min to ω max.
  • NicholsPlot[expr,{ ω , ω min, ω max}] plots expr using the variable ω .
