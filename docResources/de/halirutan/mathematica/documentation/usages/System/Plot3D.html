Plot3D

  • Plot3D[f,{x,xmin,xmax},{y,ymin,ymax}] generates a three-dimensional plot of f as a function of x and y.
  • Plot3D[{f1,f2, … },{x,xmin,xmax},{y,ymin,ymax}] plots several functions.
