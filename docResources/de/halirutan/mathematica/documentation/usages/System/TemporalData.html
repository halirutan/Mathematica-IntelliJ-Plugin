TemporalData

  • TemporalData[{s1,s2, … },tspec] temporal data with states si at times specified by tspec.
  • TemporalData[{{s11,s12, … },{s21,s22, … }, … },tspec] a temporal data collection with states sij at times specified by tspec.
  • TemporalData[{{t1,s1},{t2,s2} … }] temporal data specified by time-value pairs {ti,si}.
  • TemporalData[{{{t11,s11},{t12,s12} … },{{t21,s21},{t22,s22}, … }, … }] a temporal data collection given as lists of time-value pairs {tij,sij}.

Attributes: NHoldAll, Protected, ReadProtected

Options: Method, StateDimensions, MissingDataMethod, MetaInformation
