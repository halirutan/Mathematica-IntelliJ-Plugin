FindGeneratingFunction

  • FindGeneratingFunction[{a1,a2, … },x] attempts to find a simple generating function in x whose nth series coefficient is an.
  • FindGeneratingFunction[{{n1,a1},{n2,a2}, … },x] attempts to find a simple generating function whose nith series coefficient is ai.

Attributes: Protected, ReadProtected

Options: FunctionSpace, Method, TimeConstraint, ValidationLength
