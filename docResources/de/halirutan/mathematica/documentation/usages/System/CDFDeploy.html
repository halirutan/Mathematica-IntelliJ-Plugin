CDFDeploy

  • CDFDeploy["file.cdf",expr] deploys expr in a form that can be played by Wolfram CDF Player.
  • CDFDeploy["file.cdf",notebook] deploys a notebook.
  • CDFDeploy["file.cdf",NotebookSelection[notebook]] deploys the current selection in notebook.
  • CDFDeploy["outfile.cdf","infile.nb"] deploys the notebook "infile.nb".

Attributes: Protected, ReadProtected

Options: WindowSize, Method
