HankelMatrix

  • HankelMatrix[n] gives the n × n Hankel matrix with first row and first column being successive integers.
  • HankelMatrix[{c1,c2, … ,cn}] gives the Hankel matrix whose first column consists of elements c1, c2, … .
  • HankelMatrix[{c1,c2, … ,cm},{r1,r2, … , rn}] gives the Hankel matrix with elements ci down the first column, and ri across the last row.
