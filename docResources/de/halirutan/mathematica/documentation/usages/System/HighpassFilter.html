HighpassFilter

  • HighpassFilter[data, ω c] applies a highpass filter with a cutoff frequency ω c to an array of data.
  • HighpassFilter[data, ω c,n] uses a filter kernel of length n.
  • HighpassFilter[data, ω c,n,wfun] applies a smoothing window wfun to the filter kernel.

Attributes: Protected, ReadProtected

Options: Padding, SampleRate
