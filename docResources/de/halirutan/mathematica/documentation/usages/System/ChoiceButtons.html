ChoiceButtons

  • ChoiceButtons[] represents a pair of OK and Cancel buttons that close a dialog.
  • ChoiceButtons[{actok,actcancel}] represents OK and Cancel buttons that evaluate the corresponding acti when clicked.
  • ChoiceButtons[{lblok,lblcancel},{actok,actcancel}] uses the lbli to label the buttons.

Attributes: HoldAll, Protected, ReadProtected

Symbol has no options.
