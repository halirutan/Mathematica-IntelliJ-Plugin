HeavisideTheta

  • HeavisideTheta[x] represents the Heaviside theta function θ (x), equal to 0 for x<0 and 1 for x>0.
  • HeavisideTheta[x1,x2, … ] represents the multidimensional Heaviside theta function which is 1 only if none of the xi are not positive.

Attributes: Listable, Orderless, Protected, ReadProtected

Symbol has no options.
