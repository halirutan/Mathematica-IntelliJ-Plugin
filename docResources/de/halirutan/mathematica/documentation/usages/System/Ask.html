Ask

  • Ask["key"] is a construct for use inside AskFunction that gives the value associated with key, or interactively asks for it if it is not already known.
  • Ask["key" → formspec] uses formspec to define how input should be requested and interpreted.
  • Ask[{key1,key2, … }] gives the values associated with all the keyi, interactively asking for any that are needed.
  • Ask[{key1->formspec1,key2->formspec2, … }] uses the formspeci to define how input for the keyi should be requested and interpreted.

Attributes:Protected, ReadProtected

Symbol has no options.
