Raster

  • Raster[{{a11,a12, … }, … }] is a two-dimensional graphics primitive which represents a rectangular array of gray cells.
  • Raster[{{{r11,g11,b11}, … }, … }] represents an array of RGB color cells.
  • Raster[{{{r11,g11,b11, α 11}, … }, … }] represents an array of color cells with opacity α ij.
  • Raster[{{{a11, α 11}, … }, … }] represents an array of gray cells with the specified opacities.

Attributes: Protected, ReadProtected

Options: ColorFunction, ColorFunctionScaling
