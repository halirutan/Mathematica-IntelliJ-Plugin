SlideView

  • SlideView[{expr1,expr2, … }] represents an object in which the expri are set up to be displayed on successive slides.
  • SlideView[{expr1,expr2, … },i] makes the ith slide be the one currently displayed.

Attributes: Protected, ReadProtected

Options: Alignment, AlignmentPoint, AnimationRunning, AppearanceElements, Background, BaselinePosition, BaseStyle, ControlPlacement, DefaultBaseStyle, Deployed, Enabled, FrameMargins, ImageMargins, ImageSize
