KarhunenLoeveDecomposition

  • KarhunenLoeveDecomposition[{a1,a2, … }] gives the Karhunen – Loeve transform {b1,b2, … } of the numerical arrays {a1,a2, … } and the transformation matrix m, returning the result in the form {{b1,b2, … },m}.
  • KarhunenLoeveDecomposition[{a1,a2, … },m] uses the inverse of the matrix m for transforming the {a1,a2, … }.
